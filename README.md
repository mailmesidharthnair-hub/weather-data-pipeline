# Automated ETL Pipeline for Job Data

##  Overview

Built an automated ETL pipeline to extract, process, and store job data in a PostgreSQL database using Python and GitHub Actions.

---

## Tech Stack

* Python
* Pandas
* PostgreSQL (Neon)
* GitHub Actions
* SQL

---

## Workflow

* Extract data using Python
* Transform and clean using Pandas
* Load into PostgreSQL database
* Automate pipeline using GitHub Actions

---

## Structure

```bash
src/            # ETL script
logs/           # Pipeline logs
.github/        # Automation workflow
requirements.txt
README.md
```

---

## Run Locally

```bash
pip install -r requirements.txt
python src/main.py
```

---

## Output

Structured job data stored in PostgreSQL, ready for analysis.
